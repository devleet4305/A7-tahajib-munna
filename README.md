# 🚀 React Dashboard (Single Component)

This is a simple React dashboard project built as a **single component**, inspired by a clean Figma-style layout. The goal was to keep things straightforward while still making something interactive and visually organized.

It includes a sidebar, dynamic content, and a responsive card layout—all handled in one place so it’s easy to understand how everything connects.

---

## ✨ What’s inside

* Sidebar with clickable navigation
* Active state switching
* Dynamic content rendering from data
* Card-based layout
* Basic user interaction (buttons)
* Clean and modern UI using Tailwind CSS

---

## 📁 Project structure

```bash
src/
 ├── App.jsx
```

Everything lives inside `App.jsx`, including the data and UI logic.

## 🧠 How it works

The app uses React’s `useState` to keep track of which sidebar item is active.
Data is stored in an array and mapped to generate both the sidebar and the cards dynamically.

Clicking on different items updates the UI instantly, and buttons provide simple interaction to show how events work.
